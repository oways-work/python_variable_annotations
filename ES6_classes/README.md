# ES6 Classes

## Description
This project covers the implementation and usage of ES6 classes in JavaScript. It explores core object-oriented programming concepts such as inheritance, getters and setters, static methods, computed method names, and ES6 symbols.

## Requirements
* All files are interpreted/compiled on Ubuntu 20.04 LTS using NodeJS `20.x.x` and npm `9.x.x`
* Code is verified against linting rules using ESLint
* Code is tested using Jest
* All files should end with a new line
* The required file extension is `.js`
