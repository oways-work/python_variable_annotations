# Holberton School - Web Back End

This repository contains projects for the **Web Back End** specialization track at Holberton School.
The projects cover advanced Python concepts including type annotations, asynchronous programming, pagination, caching, and more.

## Projects

| Directory | Description |
|Args|Description|
|---|---|
| [`python_variable_annotations/`](./python_variable_annotations) | Introduction to type hints and static type checking in Python 3. |
| [`python_async_function/`](./python_async_function) | Basics of asynchronous programming using `asyncio`, coroutines, and tasks. |

## Requirements

* **OS:** Ubuntu 20.04 LTS
* **Language:** Python 3.9
* **Style:** `pycodestyle` (version 2.5.x)
