# ES6 Basics

## Description
This project covers the fundamentals of ES6 (ECMAScript 2015). It focuses on modern JavaScript syntax, including block-scoping, arrow functions, rest and spread parameters, template literals, and enhanced object properties.

## Requirements
* **Operating System:** Ubuntu 20.04 LTS
* **Node.js version:** 20.x.x
* **npm version:** 9.x.x
* **Allowed Editors:** vi, vim, emacs, Visual Studio Code
* **Testing Framework:** Jest
* **Linting:** ESLint
