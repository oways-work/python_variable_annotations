# Node JS Basics

This project introduces the fundamentals of Node.js. It covers:
* Executing basic Javascript using Node.js
* Using `process.stdin` and `process.stdout`
* Reading files synchronously and asynchronously using the `fs` module
* Creating basic HTTP servers using the `http` module
* Creating HTTP servers using the `Express` framework
* Organizing routes and controllers using ES6 Modules and Babel

## Requirements
* OS: Ubuntu 20.04 LTS
* Node.js version: 20.x.x
* Editors: vi, vim, emacs, Visual Studio Code

## Installation
Run `npm install` to install all dependencies specified in the `package.json` file.
